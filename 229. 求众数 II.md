# [229. 求众数 II](https://leetcode-cn.com/problems/majority-element-ii/)

给定一个大小为 *n* 的数组，找出其中所有出现超过 `⌊ n/3 ⌋` 次的元素。

**说明:** 要求算法的时间复杂度为 O(n)，空间复杂度为 O(1)。

**示例 1:**

```
输入: [3,2,3]
输出: [3]
```

**示例 2:**

```
输入: [1,1,1,3,3,2,2,2]
输出: [1,2]
```



```java
class Solution {
public:
    vector<int> majorityElement(vector<int>& nums) {
        int n = nums.size() / 3;
        
        vector<int> result;
        map<int, int> map;
        
        for(int val : nums)
        {
            map[val]++;
        }
        
        int res;
        auto it = map.begin();
        while(it != map.end())
        {
            if(it->second > n)
            {
                result.push_back(it->first);
            }
               
            ++it;
        }
        
        return result;
    }
};
```
